## Быстрый старт

1.  **Склонируйте репозиторий.**

    ```sh
    git clone https://github.com/mnenie/gagarin-hack.git
    ```

2.  **Начните просмотр.**

    Перейдите в директорию и запустите проект.

    ```sh
    cd nuxt-markdown-blog-starter/
    npm install
    npm run dev
    ```

3.  **Запуск!**

    Сайт запущен на `http://localhost:3000`
